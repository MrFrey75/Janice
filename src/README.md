# Janice

Janice is a multi-interface AI assistant supporting CLI, GUI, and core logic in C#/.NET 8.0.

## Project Structure
- Janice.CLI: Command-line interface
- Janice.Gui: Graphical user interface (Avalonia)
- Janice.Core: Core logic and services
